# E1005: Expected Type

A type annotation was expected but not found.

## Example

```sigil
@process (x: ) -> int = x    // Error: expected type after ':'
```

## Explanation

Type annotations follow a colon (`:`) in function parameters, return types,
and explicit variable bindings. This error occurs when the type is missing.

## Common Causes

1. **Missing parameter type:**
   ```sigil
   // Wrong
   @add (a: , b: int) -> int = a + b

   // Right
   @add (a: int, b: int) -> int = a + b
   ```

2. **Missing return type:**
   ```sigil
   // Wrong
   @get_value () -> = 42

   // Right
   @get_value () -> int = 42
   ```

3. **Missing type in binding:**
   ```sigil
   // Wrong
   let x: = 42

   // Right
   let x: int = 42
   // Or let type inference work
   let x = 42
   ```

## Solutions

1. **Add the missing type:**
   ```sigil
   @example (value: int) -> str = str(value)
   ```

2. **Use type inference when possible:**
   ```sigil
   // Type is inferred as int
   let x = 42

   // Type is inferred as str
   let name = "Alice"
   ```

3. **Common types:**
   ```sigil
   // Primitives
   let a: int = 1
   let b: float = 1.0
   let c: bool = true
   let d: str = "hello"

   // Collections
   let list: [int] = [1, 2, 3]
   let map: {str: int} = {"a": 1}

   // Generics
   let maybe: Option<int> = Some(42)
   let result: Result<str, Error> = Ok("success")
   ```
