# E2001: Type Mismatch

An expression has a different type than expected in the given context.

## Example

```sigil
@add (a: int, b: int) -> int = a + b

@main () -> void = run(
    let x: str = add(a: 1, b: 2)    // Error: expected str, found int
)
```

## Explanation

The variable `x` is declared with type `str`, but `add()` returns `int`.
Sigil is statically typed with no implicit conversions, so you must explicitly
convert between types.

## Common Causes

1. **Wrong return type annotation:**
   ```sigil
   // The function returns int, but we're assigning to str
   let x: str = add(a: 1, b: 2)
   ```

2. **Forgetting to convert types:**
   ```sigil
   // Wrong: can't add int and str
   let result = 42 + "hello"
   ```

3. **Returning wrong type from function:**
   ```sigil
   @get_age () -> str = run(    // Error: function returns str but body is int
       42
   )
   ```

## Solutions

1. **Change the variable type:**
   ```sigil
   let x: int = add(a: 1, b: 2)
   ```

2. **Use explicit type conversion:**
   ```sigil
   let x: str = str(add(a: 1, b: 2))
   ```

3. **Fix the return type:**
   ```sigil
   @get_age () -> int = run(
       42
   )
   ```

## Common Type Conversions

- `int(x)` - Convert to integer
- `float(x)` - Convert to float
- `str(x)` - Convert to string
- `byte(x)` - Convert to byte
