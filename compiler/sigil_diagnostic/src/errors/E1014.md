# E1014: Reserved Built-in Function Name

A user-defined function uses a name reserved for built-in functions.

## Example

```sigil
@print (x: int) -> void = run(    // Error: 'print' is a reserved name
    // ...
)
```

## Explanation

Certain function names are reserved for Sigil's built-in functions and cannot
be used for user-defined functions. These names are reserved **in call position**
only, meaning you can use them as variable names.

## Reserved Names

### Type Conversions
- `int`, `float`, `str`, `byte`

### Collection Operations
- `len`, `is_empty`

### Option/Result Checks
- `is_some`, `is_none`, `is_ok`, `is_err`

### Assertions
- `assert`, `assert_eq`, `assert_ne`
- `assert_some`, `assert_none`, `assert_ok`, `assert_err`
- `assert_panics`, `assert_panics_with`

### Other
- `compare`, `min`, `max`, `print`, `panic`

## What's Allowed

```sigil
// OK - as variables
let min = 5
let max = 10
let print = false

// OK - calling built-in
print(msg: "hello")
let x = min(left: a, right: b)

// NOT OK - as function names
@min (a: int, b: int) -> int = ...    // Error
@print (msg: str) -> void = ...       // Error
```

## Solutions

1. **Choose a different function name:**
   ```sigil
   // Wrong
   @min (a: int, b: int) -> int = if a < b then a else b

   // Right
   @minimum (a: int, b: int) -> int = if a < b then a else b
   @find_min (a: int, b: int) -> int = if a < b then a else b
   ```

2. **Use the built-in function:**
   ```sigil
   // Built-in min already exists
   let smallest = min(left: a, right: b)
   ```

3. **For custom print, use a descriptive name:**
   ```sigil
   // Wrong
   @print (x: int) -> void = ...

   // Right
   @print_formatted (x: int) -> void = ...
   @debug_print (x: int) -> void = ...
   ```
