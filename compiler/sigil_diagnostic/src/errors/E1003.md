# E1003: Unclosed Delimiter

A bracket, parenthesis, or brace was opened but never closed.

## Example

```sigil
@main () -> void = run(
    let list = [1, 2, 3    // Error: unclosed '['
)
```

## Explanation

Delimiters in Sigil must always be balanced:
- `(` must have a matching `)`
- `[` must have a matching `]`
- `{` must have a matching `}`

## Solutions

1. **Add the missing closing delimiter:**
   ```sigil
   // Wrong
   let list = [1, 2, 3

   // Right
   let list = [1, 2, 3]
   ```

2. **Check for nested delimiters:**
   ```sigil
   // Wrong - missing ) for inner call
   print(msg: str(add(a: 1, b: 2)

   // Right
   print(msg: str(add(a: 1, b: 2)))
   ```

3. **Use proper formatting to spot mismatches:**
   ```sigil
   // Hard to spot the error
   let result = foo(bar(baz(x, y), z)

   // Easier with formatting
   let result = foo(
       bar(
           baz(x, y),
           z,
       ),    // <- now we can see foo( needs closing
   )
   ```

## Tips

- Most editors highlight matching delimiters
- Use auto-formatting (`sigil fmt`) to catch these issues early
- Count opening and closing delimiters if you're stuck
