# E2002: Unknown Type

A type name was used that doesn't exist in the current scope.

## Example

```sigil
@process (x: Foo) -> int = 42    // Error: unknown type 'Foo'
```

## Explanation

Type names must be defined before they can be used. This error occurs when
you reference a type that:
- Hasn't been defined
- Is misspelled
- Hasn't been imported

## Common Causes

1. **Typo in type name:**
   ```sigil
   // Wrong
   let x: Stirng = "hello"

   // Right
   let x: str = "hello"
   ```

2. **Missing import:**
   ```sigil
   // Wrong - Duration not imported
   @wait (time: Duration) -> void = ...

   // Right
   use std.time { Duration }
   @wait (time: Duration) -> void = ...
   ```

3. **Type not defined:**
   ```sigil
   // Wrong - User not defined
   @get_user (id: int) -> User = ...

   // Right - define the type first
   type User = { id: int, name: str }
   @get_user (id: int) -> User = ...
   ```

## Built-in Types

### Primitives
- `int`, `float`, `bool`, `str`, `char`, `byte`, `void`, `Never`

### Special
- `Duration`, `Size`

### Collections
- `[T]` (list), `{K: V}` (map), `Set<T>`

### Compound
- `(T, U)` (tuple), `()` (unit), `(T) -> U` (function)

### Standard
- `Option<T>`, `Result<T, E>`, `Error`, `Ordering`, `Range<T>`

## Solutions

1. **Check spelling:**
   ```sigil
   // Common mistakes
   Stirng -> str
   Int -> int
   Boolean -> bool
   ```

2. **Import the type:**
   ```sigil
   use std.time { Duration }
   use std.collections { Set }
   ```

3. **Define custom types:**
   ```sigil
   type User = { id: int, name: str }
   type Status = Active | Inactive | Pending
   ```
