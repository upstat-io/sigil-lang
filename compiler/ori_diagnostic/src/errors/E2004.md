# E2004: Argument Count Mismatch

A function was called with the wrong number of arguments.

## Example

```ori
@add (a: int, b: int) -> int = a + b

@main () -> void = run(
    let sum = add(a: 1)    // Error: expected 2 arguments, found 1
)
```

## Explanation

Functions in Ori require exactly the number of arguments specified in their
signature. Unlike some languages, Ori doesn't support optional parameters
or variable arguments in user-defined functions.

## Solutions

1. **Provide all required arguments:**
   ```ori
   // Wrong - missing b
   let sum = add(a: 1)

   // Right
   let sum = add(a: 1, b: 2)
   ```

2. **Remove extra arguments:**
   ```ori
   // Wrong - too many arguments
   let sum = add(a: 1, b: 2, c: 3)

   // Right
   let sum = add(a: 1, b: 2)
   ```

3. **Check the function signature:**
   ```ori
   // If you're not sure what arguments are needed,
   // check the function definition
   @greet (name: str, greeting: str) -> str = greeting + ", " + name

   // Call with both required arguments
   let msg = greet(name: "World", greeting: "Hello")
   ```
