# E1009: Missing Pattern Argument

A required argument is missing from a pattern call.

## Example

```ori
@main () -> void = run(
    recurse(base: 1, step: self(n - 1))    // Error: missing 'condition' argument
)
```

## Explanation

Built-in patterns like `recurse`, `parallel`, and `timeout` require specific
arguments. This error occurs when a required argument is not provided.

## Required Arguments by Pattern

| Pattern | Required Arguments |
|---------|-------------------|
| `recurse` | `condition`, `base`, `step` |
| `parallel` | `tasks` |
| `spawn` | `tasks` |
| `timeout` | `op`, `after` |
| `cache` | `key`, `op` |
| `with` | `acquire`, `use` |

## Common Causes

1. **Missing required argument in `recurse`:**
   ```ori
   // Wrong - missing condition
   recurse(base: 1, step: self(n - 1))

   // Right
   recurse(
       condition: n <= 0,
       base: 1,
       step: n * self(n - 1)
   )
   ```

2. **Missing required argument in `timeout`:**
   ```ori
   // Wrong - missing after
   timeout(op: slow_operation())

   // Right
   timeout(op: slow_operation(), after: 5s)
   ```

3. **Missing required argument in `parallel`:**
   ```ori
   // Wrong - missing tasks
   parallel(max_concurrent: 4)

   // Right
   parallel(tasks: [task1, task2, task3], max_concurrent: 4)
   ```

## Solutions

1. **Add all required arguments:**
   ```ori
   recurse(
       condition: n <= 1,
       base: 1,
       step: n * self(n - 1)
   )
   ```

2. **Check pattern documentation for required vs optional:**
   ```ori
   // Required: tasks
   // Optional: max_concurrent, timeout
   parallel(
       tasks: work_items,
       max_concurrent: 4    // optional
   )
   ```
