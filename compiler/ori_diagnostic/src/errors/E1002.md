# E1002: Expected Expression

An expression was expected but not found.

## Example

```ori
@main () -> void = run(
    let x = if then 1    // Error: expected expression after 'if'
)
```

## Explanation

Many constructs in Ori require an expression in a particular position. This
error occurs when the parser expects to find an expression but finds something
else instead.

## Common Causes

1. **Missing condition in `if`:**
   ```ori
   // Wrong
   if then 1 else 0

   // Right
   if true then 1 else 0
   ```

2. **Missing operand in binary expression:**
   ```ori
   // Wrong
   let x = 1 +

   // Right
   let x = 1 + 2
   ```

3. **Empty binding value:**
   ```ori
   // Wrong
   let x =

   // Right
   let x = 42
   ```

4. **Missing function body:**
   ```ori
   // Wrong
   @add (a: int, b: int) -> int =

   // Right
   @add (a: int, b: int) -> int = a + b
   ```

## Solutions

1. **Add the missing expression:**
   ```ori
   let condition = x > 0
   let result = if condition then "positive" else "non-positive"
   ```

2. **Complete the binary operation:**
   ```ori
   let sum = a + b
   ```

3. **For empty values, use the appropriate literal:**
   ```ori
   let empty_list: [int] = []
   let empty_map: {str: int} = {}
   let nothing: Option<int> = None
   ```
