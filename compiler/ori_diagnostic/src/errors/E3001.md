# E3001: Unknown Pattern

A pattern name was used that doesn't exist.

## Example

```ori
@main () -> void = run(
    let result = unknown_pattern(x: 42)    // Error: unknown pattern
)
```

## Explanation

Ori has a fixed set of built-in patterns (like `run`, `try`, `match`, etc.).
If you use a name that looks like a pattern call but isn't recognized, you'll
get this error.

## Built-in Patterns

**Sequential (`function_seq`):**
- `run(...)` - Sequential evaluation
- `try(...)` - Error propagation with `?`
- `match(value, arms...)` - Pattern matching
- `catch(expr)` - Catch panics

**Named (`function_exp`):**
- `recurse(condition:, base:, step:, ...)` - Recursive computation
- `parallel(tasks:, ...)` - Concurrent execution
- `spawn(tasks:, ...)` - Fire-and-forget tasks
- `timeout(op:, after:)` - Timed operations
- `cache(key:, op:, ...)` - Memoization
- `with(acquire:, use:, release:)` - Resource management
- `for(over:, match:, ...)` - Iteration

## Solutions

1. **Check spelling:**
   ```ori
   // Wrong
   let x = runn(let a = 1, a)

   // Right
   let x = run(let a = 1, a)
   ```

2. **Use the correct pattern:**
   ```ori
   // If you want error handling, use try not catch
   let result = try(
       let value = might_fail()?,
       Ok(value)
   )
   ```

3. **For stdlib functions, use method syntax:**
   ```ori
   // map, filter, fold are methods, not patterns
   let doubled = items.map(transform: x -> x * 2)
   ```
