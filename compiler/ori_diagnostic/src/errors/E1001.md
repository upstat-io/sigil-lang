# E1001: Unexpected Token

The parser encountered a token that doesn't make sense in the current context.

## Example

```ori
@main () -> void = run(
    let x = 42 42    // Error: unexpected `42`, expected operator or newline
)
```

## Explanation

This error occurs when the parser expects one kind of token but finds something
else entirely. Common causes include:

- Missing operators between expressions
- Typos in keywords
- Mismatched delimiters
- Extra or missing punctuation

## Solutions

1. **Add missing operator:**
   ```ori
   // Wrong
   let x = 42 42

   // Right - if you meant addition
   let x = 42 + 42

   // Right - if you meant a list
   let x = [42, 42]
   ```

2. **Check for typos in keywords:**
   ```ori
   // Wrong
   lat x = 42    // 'lat' is not 'let'

   // Right
   let x = 42
   ```

3. **Check delimiters match:**
   ```ori
   // Wrong
   let list = [1, 2, 3)    // mismatched [ and )

   // Right
   let list = [1, 2, 3]
   ```

4. **Remove extra punctuation:**
   ```ori
   // Wrong
   let x = 42,;

   // Right
   let x = 42
   ```
