# Yin and yang

**Problem:** One well-known symbol of the philosophy of duality known as yin and yang is the taijitu.

**Requirements:**
- Create a function that, given a parameter representing size, generates such a symbol scaled to the requested size.
- Generate and display the symbol for two different (small) sizes.

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
