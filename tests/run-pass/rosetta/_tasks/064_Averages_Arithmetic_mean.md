# Averages/Arithmetic mean

**Problem:** Calculate the arithmetic mean (average) of a numeric vector.

**Requirements:**
- Sum all values in the vector
- Divide by the count of elements
- Handle zero-length input appropriately (return 0, NaN, or undefined)

**Success Criteria:**
- mean([1, 2, 3, 4, 5]) = 3.0
- mean([3, 1, 4, 1, 5, 9]) = 3.8333...
- Empty vector handling matches language conventions
