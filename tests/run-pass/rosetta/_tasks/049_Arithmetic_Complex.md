# Arithmetic/Complex

**Problem:** Implement complex number arithmetic operations for numbers in the form a + bi.

**Requirements:**
- Implement as separate functions:
  - Addition: (a+bi) + (c+di) = (a+c) + (b+d)i
  - Multiplication: (a+bi)(c+di) = (ac-bd) + (ad+bc)i
  - Negation: -(a+bi) = -a - bi
  - Inversion: 1/(a+bi) = (a-bi)/(a²+b²)
- Optional: Conjugation: conj(a+bi) = a-bi
- Print results for each operation

**Success Criteria:**
- Test with z₁ = 1 + 1i and z₂ = 3.14159 + 1.2i
- All operations produce mathematically correct results
