# Last Friday of each month

**Problem:** Write a program or a script that returns the date of the last Fridays of each month of a given year. The year may be given through any simple input method in your language (command line, std in, etc). Example of an expected output: ./last_fridays 2012 Related tasks

**Requirements:**
- Five weekends
- Day of the week
- Find the last Sunday of each month

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
