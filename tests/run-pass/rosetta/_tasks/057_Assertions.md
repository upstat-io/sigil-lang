# Assertions

**Problem:** Demonstrate how assertions work by asserting that a variable equals 42.

**Requirements:**
- Create an assertion that checks if an integer variable equals 42
- Show the basic assertion syntax for the language
- Optionally demonstrate custom error messages
- Illustrate behavior when the assertion fails

**Success Criteria:**
- Assertion passes when variable equals 42
- Assertion fails/throws when variable does not equal 42
- Demonstrates language's idiomatic assertion mechanism
