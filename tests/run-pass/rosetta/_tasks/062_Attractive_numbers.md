# Attractive numbers

**Problem:** Find numbers whose count of prime factors (with multiplicity) is itself prime.

**Requirements:**
- Count prime factors with multiplicity (e.g., 20 = 2 × 2 × 5 has 3 factors)
- A number is attractive if its prime factor count is prime
- Find all attractive numbers from 1 to 120

**Success Criteria:**
- Output 74 attractive numbers
- Includes: 4, 6, 8, 9, 10, 12, 14, 15, 18, 20...
- Example: 20 has 3 prime factors, 3 is prime, so 20 is attractive
