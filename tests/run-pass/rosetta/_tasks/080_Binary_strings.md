# Binary strings

**Problem:** Implement functions to handle binary strings (arbitrary byte sequences).

**Requirements:**
- String creation and destruction (memory management)
- String assignment (copy references or values)
- String comparison (lexicographic or equality)
- String cloning/copying (deep copy)
- Empty string check
- Append byte to string
- Extract substring
- Replace all occurrences of a pattern
- Join/concatenate strings

**Success Criteria:**
- All operations handle arbitrary binary data including null bytes
- Operations work correctly for edge cases (empty strings, single bytes)
