# Box the compass

**Problem:** Convert compass headings in degrees to 32-point compass direction names.

**Requirements:**
- Accept heading in degrees (0-360)
- Return appropriate 32-point compass point name
- Test with 33 specific headings: 0.0, 16.87, 16.88, 33.75, ... 354.38
- Display table with index (1-32), compass point name, and degree value

**Success Criteria:**
- 0.0° → "North"
- 90.0° → "East"
- 180.0° → "South"
- All 32 compass points correctly identified
