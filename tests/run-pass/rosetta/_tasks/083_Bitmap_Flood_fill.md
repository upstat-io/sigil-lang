# Bitmap/Flood fill

**Problem:** Implement flood fill algorithm to fill connected regions with a color.

**Requirements:**
- Fill an area starting from a seed point
- Use color banks to define contained area or target color
- Support recursive, iterative (queue/stack), or scanline approaches
- Optional: add tolerance parameter for color matching

**Success Criteria:**
- Successfully fills connected regions of same color
- Stops at boundaries (different color or image edge)
- Can fill both interior and exterior regions
