# Bernoulli numbers

**Problem:** Compute Bernoulli numbers B₀ through B₆₀ using the Akiyama-Tanigawa algorithm.

**Requirements:**
- Display Bernoulli numbers B₀ through B₆₀
- Suppress output of zero values (all odd Bernoulli numbers except B₁ are zero)
- Express as reduced fractions
- Index each number for identification

**Success Criteria:**
- B(0) = 1/1
- B(1) = 1/2
- B(2) = 1/6
- B(60) = -1215233140483755572040304994079820246041491/56786730
