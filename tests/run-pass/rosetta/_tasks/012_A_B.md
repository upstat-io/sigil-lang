# A+B

**Problem:** Classic programming contest problem - read two integers and output their sum.

**Requirements:**
- Read two integers from input (space-separated)
- Constraints: -1000 ≤ A, B ≤ +1000
- Output their sum

**Success Criteria:**
- Input: `2 2` → Output: `4`
- Input: `3 2` → Output: `5`
