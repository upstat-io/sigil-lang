# Ascending primes

**Problem:** Generate all primes with strictly ascending decimal digits.

**Requirements:**
- Find all prime numbers where each digit is strictly greater than the previous
- Digits must be in strictly ascending order (no duplicates, no decreasing)
- Display all such primes in ascending numerical order
- Efficient approach: generate ascending digit candidates first, then test primality

**Success Criteria:**
- Output exactly 100 ascending primes
- Starts with: 2, 3, 5, 7, 13, 17, 19, 23, 29, 37...
- Ends with: ...12356789, 23456789
