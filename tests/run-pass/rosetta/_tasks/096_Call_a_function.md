# Call a function

**Problem:** Demonstrate different syntax and semantics for calling functions.

**Requirements:**
- Call a function with no arguments
- Call a function with fixed number of arguments
- Call a function with optional arguments
- Call a function with variable number of arguments
- Call a function with named arguments
- Use a function in expression context
- Obtain the return value of a function
- Distinguish pass by value vs pass by reference
- Show partial application if supported

**Success Criteria:**
- Demonstrate all applicable calling conventions for the language
- Show both built-in and user-defined function calls
