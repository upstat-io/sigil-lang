# Floyd's triangle

**Problem:** Floyd's triangle lists the natural numbers in a right triangle aligned to the left where The first few lines of a Floyd triangle looks like this: 11 12 13 14 15

**Requirements:**
- successive rows start towards the left with the next number followed by successive naturals listing one more number than the line above.
- Write a program to generate and display here the first n lines of a Floyd triangle. (Use n=5 and n=14 rows).
- Ensure that when displayed in a mono-space font, the numbers line up in vertical columns as shown and that only one space separates numbers of the last row.

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
