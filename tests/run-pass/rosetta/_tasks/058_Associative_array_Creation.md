# Associative array/Creation

**Problem:** Create an associative array (dictionary, map, or hash).

**Requirements:**
- Create an empty or initialized associative array
- Add key-value pairs to the structure
- Retrieve values using keys
- Optionally: delete key-value pairs
- Optionally: check for key existence

**Success Criteria:**
- Demonstrate creation of key-value mapping structure
- Show adding and retrieving entries by key
- Use language's native associative array type
