# Sequence of non-squares

**Problem:** Show that the following remarkable formula gives the sequence of non-square natural numbers: n + floor(1/2 + sqrt(n)) This is sequence A000037 in the OEIS database.

**Requirements:**
- Print out the values for n in the range 1 to 22
- Show that no squares occur for n less than one million

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
