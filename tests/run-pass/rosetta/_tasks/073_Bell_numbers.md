# Bell numbers

**Problem:** Generate Bell numbers which count the ways to partition a set of n elements.

**Requirements:**
- Implement Bell number generation (using Bell triangle or Stirling numbers)
- Display at least the first 15 Bell numbers
- Display the 50th Bell number (if big integers supported)
- Optionally show the first 10 rows of Bell's triangle

**Success Criteria:**
- First 15 Bell numbers: 1, 1, 2, 5, 15, 52, 203, 877, 4140, 21147, 115975, 678570, 4213597, 27644437, 190899322
- B₅₀ = 10726137154573358400342215518590002633917247281
