# Bitmap

**Problem:** Implement basic storage for RGB raster graphics images.

**Requirements:**
- Allocation function: create uninitialized image given width and height
- Fill function: fill image with a plain RGB color
- Set pixel function: set a given pixel to a specific color
- Get pixel function: retrieve color value of a pixel
- Handle RGB (red, green, blue) color values

**Success Criteria:**
- Can create images of arbitrary dimensions
- Can fill, set, and get pixel colors
- Serves as foundation for raster graphics operations
