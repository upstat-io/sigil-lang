# Brazilian numbers

**Problem:** Find numbers that have a representation in some base with all identical digits.

**Requirements:**
- A number N is Brazilian if it equals "ddd...d" in some base B (1 < B < N-1)
- Display first 20 Brazilian numbers
- Display first 20 odd Brazilian numbers
- Display first 20 prime Brazilian numbers

**Success Criteria:**
- First 20 Brazilian: 7, 8, 10, 12, 13, 14, 15, 16, 18, 20, 21, 22, 24, 26, 27, 28, 30, 31, 32, 33
- First 20 odd Brazilian: 7, 13, 15, 21, 27, 31, 33, 35, 39, 43...
- First 20 prime Brazilian: 7, 13, 31, 43, 73, 127, 157, 211...
- Example: 7 = "111" in base 2
