# Calculating the value of e

**Problem:** Calculate Euler's number e using series summation.

**Requirements:**
- Compute e using Taylor series: e = Σ(1/n!) for n from 0 to infinity
- Or use: e = 1 + 1/1! + 1/2! + 1/3! + 1/4! + ...
- Achieve reasonable precision (15+ decimal places for double precision)
- Continue summing until convergence

**Success Criteria:**
- e ≈ 2.718281828459045
- Typical convergence requires 15-20 iterations for double precision
