# Anti-primes

**Problem:** Generate highly composite numbers (anti-primes) - natural numbers with more factors than any smaller number.

**Requirements:**
- Find natural numbers that have more divisors than all smaller positive integers
- Generate the first twenty anti-primes
- Count divisors efficiently

**Success Criteria:**
- Output: 1, 2, 4, 6, 12, 24, 36, 48, 60, 120, 180, 240, 360, 720, 840, 1260, 1680, 2520, 5040, 7560
- The 20th anti-prime (7560) has 64 divisors
