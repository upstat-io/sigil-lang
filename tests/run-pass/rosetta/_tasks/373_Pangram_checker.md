# Pangram checker

**Problem:** A pangram is a sentence that contains all the letters of the English alphabet at least once. For example: The quick brown fox jumps over the lazy dog. Write a function or method to check a sentence to see if it is a pangram (or not) and show its use. Related tasks:

**Requirements:**
- determine if a string has all the same characters
- determine if a string has all unique characters

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
