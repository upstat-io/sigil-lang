# Associative array/Iteration

**Problem:** Iterate over key-value pairs in an associative array and print each pair.

**Requirements:**
- Iterate over complete key-value pairs and print them
- If available, iterate over just the keys
- If available, iterate over just the values
- Demonstrate language-specific iteration idioms

**Success Criteria:**
- Print all key-value pairs from the associative array
- Separately iterate and print keys only
- Separately iterate and print values only
