# Binary digits

**Problem:** Convert non-negative integers to their binary (base-2) representation.

**Requirements:**
- Display binary digits for given non-negative integers
- No leading zeros (except for input 0)
- No radix or sign markers
- Output only binary digits followed by newline

**Success Criteria:**
- 5 → 101
- 50 → 110010
- 9000 → 10001100101000
- 0 → 0
