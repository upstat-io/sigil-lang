# Phrase reversals

**Problem:** Given a string of space separated words containing the following phrase: rosetta code phrase reversal Show your output here.

**Requirements:**
- Reverse the characters of the string.
- Reverse the characters of each individual word in the string, maintaining original word order within the string.
- Reverse the order of each word of the string, maintaining the order of characters in each word.

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
