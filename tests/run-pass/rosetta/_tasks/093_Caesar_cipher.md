# Caesar cipher

**Problem:** Implement a Caesar cipher for encoding and decoding text.

**Requirements:**
- Rotate letters by a key value (1-25)
- Wrap Z to A (and z to a)
- Preserve non-letter characters
- Support both encoding and decoding
- Key 13 is equivalent to ROT-13

**Success Criteria:**
- Key 2: "HI" → "JK"
- Key 20: "HI" → "BC"
- Decoding reverses encoding with same key
