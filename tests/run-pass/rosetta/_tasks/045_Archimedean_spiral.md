# Archimedean spiral

**Problem:** Draw an Archimedean spiral using the polar equation r = a + b·θ.

**Requirements:**
- Implement the spiral equation: r = a + b·θ where r is radial distance, θ is angle
- Convert from polar to Cartesian coordinates: x = r·cos(θ), y = r·sin(θ)
- Render the spiral visually (graphical output or ASCII art)
- Allow parameters a and b to control the spiral shape

**Success Criteria:**
- Output displays a recognizable Archimedean spiral
- Spiral expands outward at a constant rate as angle increases
