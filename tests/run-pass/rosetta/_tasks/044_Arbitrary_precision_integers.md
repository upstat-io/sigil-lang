# Arbitrary-precision integers

**Problem:** Calculate 5^(4^(3^2)) using built-in arbitrary-precision integer capabilities.

**Requirements:**
- Use language-native arbitrary-precision arithmetic (not a custom implementation)
- Calculate 5 to the power of 262144 (which is 4^(3^2))
- Display the first and last 20 digits of the result
- Count and display the total number of digits

**Success Criteria:**
- First 20 digits: 62060698786608744707
- Last 20 digits: 92256259918212890625
- Total digits: 183231
