# Lucas-Lehmer test

**Problem:** Lucas-Lehmer Test: for p an odd prime, the Mersenne number 2^p-1 is prime if and only if 2^p-1 divides S(p-1) where S(n+1)=(S(n))^2-2, and S(1)=4. Calculate all Mersenne primes up to the implementation's maximum precision, or the 47th Mersenne prime (whichever comes first).

**Requirements:**
- Calculate all Mersenne primes up to the implementation's maximum precision, or the 47th Mersenne prime (whichever comes first).
