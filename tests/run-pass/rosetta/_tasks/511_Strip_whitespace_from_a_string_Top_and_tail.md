# Strip whitespace from a string/Top and tail

**Problem:** Demonstrate how to strip leading and trailing whitespace from a string. The solution should demonstrate how to achieve the following three results: For the purposes of this task whitespace includes non printable characters such as the space character, the tab character, and other such characters that have no corresponding graphical representation.

**Requirements:**
- String with leading whitespace removed
- String with trailing whitespace removed
- String with both leading and trailing whitespace removed

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
