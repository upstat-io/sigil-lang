# Optional parameters

**Problem:** Define a function/method/subroutine which sorts a sequence ("table") of sequences ("rows") of strings ("cells"), by one of the strings. Besides the input to be sorted, it shall have the following optional parameters: A function specifying the ordering of strings; lexicographic by default. An integer specifying which string of each row to compare; the first by default. Reverses the ordering.

**Requirements:**
- Named Arguments

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
