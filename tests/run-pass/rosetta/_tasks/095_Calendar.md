# Calendar

**Problem:** Generate a text calendar for any year.

**Requirements:**
- Display all 12 months in formatted layout
- Show day-of-week headers (Su Mo Tu We Th Fr Sa)
- Center the year in output
- Arrange months across page width (typically 3 per row for 80 chars)
- Handle leap years correctly
- Support variable page widths (minimum 20 characters)

**Success Criteria:**
- Test case: year 1969
- Proper weekday alignment for first day of each month
- Correct number of days per month including leap years
