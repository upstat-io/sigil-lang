# Determine if a string has all unique characters

**Problem:** Given a character string (which may be empty, or have a length of zero characters): Use (at least) these five test values (strings): 1234567890ABCDEFGHIJKLMN0PQRSTUVWXYZ Show all output here on this page.

**Requirements:**
- create a function/procedure/routine to:
- determine if all the characters in the string are unique
- indicate if or which character is duplicated and where
- display each string and its length (as the strings are being examined)
- a zero─length (empty) string shall be considered as unique
- process the strings from left─to─right
- if unique, display a message saying such
- if not unique, then:
- display a message saying such
- display what character is duplicated

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
