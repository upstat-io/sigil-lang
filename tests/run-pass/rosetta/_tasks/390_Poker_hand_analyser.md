# Poker hand analyser

**Problem:** Create a program to parse a single five card poker hand and rank it according to this list of poker hands. A poker hand is specified as a space separated list of five playing cards. Each input card has two characters indicating face and suit. 2d (two of diamonds). Faces are: a, 2, 3, 4, 5, 6, 7, 8, 9, 10, j, q, k Suits are: h (hearts), d (diamonds), c (clubs), and s (spades), or alternatively, the unicode card-suit characters: ♥ ♦ ♣ ♠ Duplicate cards are illegal.

**Requirements:**
- use the playing card characters introduced with Unicode 6.0 (U+1F0A1 - U+1F0DE).
- allow two jokers
- use the symbol joker
- five-of-a-kind would then be the highest hand
- Playing cards
- Card shuffles
- Deal cards_for_FreeCell
- War Card_Game
- Go Fish

**Success Criteria:**
- Program produces correct output for test cases
- Implementation matches Rosetta Code specification
